# OpenParrot MT
Hyperspecialised fork of OpenParrot for Wangan Midnight Maximum Tune.  
Based on [OpenParrot Asakura](https://github.com/ProjectAsakura/OpenParrot).  

> Note: This is a branch customized for a specific server's launcher.  

## Supported Games

> Note: The revision may not be accurate, but only public dumps will be supported here.  

| **Game**   | **Revision** | **Support status** |
|:--------:|:-----------:|:---------------------:|
| WMMT5    | `1.05.00` | Full |
| WMMT5DX  | `2.00.02` | No improvement |
| WMMT5DX+ | `3.05.00` | Full |
| WMMT6    | `1.03.04` | No improvement |
| WMMT6R   | `2.08.00` | No improvement |
| WMMT6RR  | `3.05.03` | Full |

**Full**: Full support, including resolution patches, fixes for critical issues, etc.  
**No improvements**: The current code is still the original implementation from when it was forked.
